# Estate Agent Property Showcase

## Project Overview
This is a React-based Estate Agent Property Showcase web application developed for coursework submission.  
The application allows users to search for properties, view property details, add favourites, and view property locations on a map.




## Features
- Property listing page
- Property details page with images, floor plan, and map
- Search functionality using:
  - Property type (House / Flat)
  - Price range (min and max)
  - Bedrooms (min and max)
  - Date added
  - Postcode area
- Favourites functionality
- Responsive user interface

---

## Technologies Used
- React (TypeScript)
- Vite
- Tailwind CSS
- shadcn/ui
- React Router
